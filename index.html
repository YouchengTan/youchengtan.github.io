Summary

  We want to implement a realistic simulation of soft bodies colliding with other soft bodies as well as hard surfaces with as realistit lighting if possible. The main goal of this project is to create visually convincing deformation simulations of colliding soft bodies and hard surfaces.

Problem Description

  There are a few main challenges:

  1. Accurate force simulation between the triangles on the mesh.
  2. Gravity simulation and initial conditions (forces, velocity)
  3. Collision detection between different bodies

Goals and Deliverables

 
Here we lay out the goals for this project:

  • Gravity simulation and arbitary initial conditions on objects of various shapes obtained from mesh files
  • Accurate simulation of soft body dynamics.
